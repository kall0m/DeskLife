# DeskLife

DeskLife is a calm health and wellbeing app designed around the reality of the working day. It helps office workers, freelancers, and other desk-based professionals build healthier routines through small, manageable actions focused on food, movement, and daily habits.

[View the live project](https://desklife.vercel.app)

## Основни възможности

- дневен план с интерактивни задачи;
- седмична статистика;
- примерни хранителни и тренировъчни планове;
- навици за по-здравословен работен ден;
- TDEE и BMI калкулации;
- запазване на резултати в браузъра;
- responsive интерфейс и dark theme.

## Технологии

Проектът използва Next.js, React, TypeScript, CSS, localStorage, GSAP, GitHub и Vercel.
